# project jeff
Project jeff is an effective, unblocked games website that runs locally on your device. The games consist of TyCo Originals, and other open source games. This project will come out in releases and is not guaranteed to be without bugs. Report any bugs to us via hello-tyco.

Project Jeff is for anyone needing a break, but with restrictions. This project allows users to host games on their own devices. This means the games will not rely on internet capabilities, so they can't be blocked by any internet restrictions. This allows you to just switch off your wi-fi, and get gaming.

Project Jeff is a project by TyCo Innovations, and shall not be used for any commercial or business purposes, it may only be used on up to 2 personal devices and be shared via the official repository for downloads. This project may not be edited to contain malware or any malicious software.
